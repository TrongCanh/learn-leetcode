## 📖 Đề bài

### Mô tả
Cho 2 strings `text1` và `text2`. Tìm **độ dài của LCS** (Longest Common Subsequence).

Subsequence = dãy ký tự giữ nguyên thứ tự nhưng không nhất thiết liền kề.

### Ví dụ

**Example 1:**
```
Input:  text1 = "abcde", text2 = "ace"
Output: 3
Giải thích: LCS = "ace"
```

**Example 2:**
```
Input:  text1 = "abc", text2 = "def"
Output: 0
```

## 🧠 Phân tích & Hướng tư duy

### 🔍 1. Hiểu đề bài (5 giây)

```
Hỏi: Độ dài LCS của 2 strings
→ Classic DP 2D
→ dp[i][j] = LCS của text1[0..i-1] và text2[0..j-1]
```

---

### 🤔 2. Tư duy từng bước

> **Aha moment:**
> ```
> if text1[i-1] === text2[j-1]:
>   dp[i][j] = dp[i-1][j-1] + 1
> else:
>   dp[i][j] = max(dp[i-1][j], dp[i][j-1])
>
> → Nếu chars bằng nhau: extend LCS trước
> → Nếu không: lấy max của 2 trường hợp
> ```

### ⚠️ 4. Common Pitfalls

// ❌ Pitfall 1: Dùng text1[i] thay vì text1[i-1]

// ❌ Pitfall 2: 1D optimization không duyệt ngược

// for (let j = n; j >= 1; j--) // ✅
```

### 🔄 5. Hướng tiếp cận

#### 🔹 Cách 1: DP 2D (O(m × n), O(m × n))

```javascript
function longestCommonSubsequence(text1, text2) {
  const m = text1.length;
  const n = text2.length;
  const dp = Array(m + 1).fill().map(() => Array(n + 1).fill(0));

  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (text1[i - 1] === text2[j - 1]) {
        dp[i][j] = dp[i - 1][j - 1] + 1;
      } else {
        dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
      }
    }
  }

  return dp[m][n];
}
```

## ✅ Đáp án cuối cùng

```javascript
/**
 * Longest Common Subsequence - DP 2D
 * Time: O(m × n) | Space: O(m × n)
 */
function longestCommonSubsequence(text1, text2) {
  const m = text1.length, n = text2.length;
  const dp = Array(m + 1).fill().map(() => Array(n + 1).fill(0));
  for (let i = 1; i <= m; i++) {
    for (let j = 1; j <= n; j++) {
      if (text1[i - 1] === text2[j - 1]) dp[i][j] = dp[i - 1][j - 1] + 1;
      else dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
    }
  }
  return dp[m][n];
}
```
