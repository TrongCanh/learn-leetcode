## 📖 Đề bài

### Mô tả
Cho một mảng số nguyên `nums`, tìm tất cả các **bộ 3 số** có tổng = 0.

**Không được trùng lặp** kết quả.

### Ví dụ

**Example 1:**
```
Input:  nums = [-1, 0, 1, 2, -1, -4]
Output: [[-1, -1, 2], [-1, 0, 1]]
Giải thích:
- (-1) + (-1) + 2 = 0
- (-1) + 0 + 1 = 0
```

**Example 2:**
```
Input:  nums = [0, 1, 1]
Output: []
```

**Example 3:**
```
Input:  nums = [0, 0, 0]
Output: [[0, 0, 0]]
```

### Constraints
```
3 <= nums.length <= 3000
-10^5 <= nums[i] <= 10^5
```

### 🤔 2. Tư duy từng bước (Step-by-step)

#### Bước 1: Nghĩ đến cách "ngây thơ" nhất

> **Hỏi:** "Làm sao tìm 3 số có tổng = 0?"

**Cách 1: Brute Force - 3 nested loops**

```
Input: [-1, 0, 1, 2, -1, -4]

Bước 1: i=0, j=1, k=2
  nums[0] + nums[1] + nums[2] = -1 + 0 + 1 = 0 ✓
  → Thêm [-1, 0, 1]

Bước 2: i=0, j=1, k=3
  = -1 + 0 + 2 = 1 ✗

Bước 3: ... tiếp tục tất cả combinations

→ O(n³) - QUÁ CHẬM!
```

**⚠️ Nhược điểm:**
```javascript
function threeSum(nums) {
  const result = [];
  for (let i = 0; i < nums.length; i++) {
    for (let j = i + 1; j < nums.length; j++) {
      for (let k = j + 1; k < nums.length; k++) {
        if (nums[i] + nums[j] + nums[k] === 0) {
          result.push([nums[i], nums[j], nums[k]]);
        }
      }
    }
  }
  return result;
}
// → O(n³) với n = 3000 → ~27 tỷ operations → CHẬM!
```

#### Bước 2: Tối ưu - Giảm thành Two Sum

> **Aha moment #1:**
> "Thay vì tìm 3 số cùng lúc, em tìm 2 số có tổng = -nums[i]!"
> "→ 3Sum = Fix 1 số + Two Sum!"

> **Aha moment #2:**
> "Two Sum có thể giải bằng Two Pointers O(n) nếu sorted!"

```
Luồng tư duy:
┌─────────────────────────────────────────────────────┐
│ nums = [-1, 0, 1, 2, -1, -4]                       │
│                                                       │
│ Sort: [-4, -1, -1, 0, 1, 2]                        │
│                                                       │
│ Fix i = 0: nums[0] = -4                            │
│   → Tìm 2 số có tổng = 4                          │
│   → left=1, right=5 → sum=-1+2=1 < 4 → left++    │
│   → left=2, right=5 → sum=-1+2=1 < 4 → left++    │
│   → ...                                           │
│                                                       │
│ Fix i = 1: nums[1] = -1                            │
│   → Tìm 2 số có tổng = 1                          │
│   → left=2, right=5 → sum=-1+2=1 → FOUND! [-1,-1,2]│
└─────────────────────────────────────────────────────┘
```

#### Bước 4: Xác định edge cases

```
Edge Case 1: nums = [0, 0, 0]
  → Sort: [0, 0, 0]
  → i=0: left=1, right=2 → sum=0+0=0 → FOUND [0,0,0]
  → return [[0, 0, 0]] ✓

Edge Case 2: nums = [0, 1, 2]
  → Sort: [0, 1, 2]
  → sum luôn > 0 → return [] ✓

Edge Case 3: nums có nhiều duplicates
  [-1, -1, -1, 0, 0, 0, 1, 1, 1]
  → Phải skip duplicates đúng cách!

Edge Case 4: nums = [-2, 0, 1, 1, 2, 2]
  → [[-2, 0, 2], [-2, 1, 1], [0, 0, 0]] (nếu có)
```

### ⚠️ 4. Common Pitfalls (Lỗi hay mắc)

```javascript
// ❌ Pitfall 1: Không sort trước
function threeSumWrong(nums) {
  const result = [];
  for (let i = 0; i < nums.length; i++) {
    // ... tìm 2 số
  }
  return result;
}
// → Kết quả có thể trùng lặp!

// ✅ Đúng: Sort trước
nums.sort((a, b) => a - b);

// ❌ Pitfall 2: Không skip duplicates
function threeSumNoSkip(nums) {
  // ... tìm được [-1, 0, 1]
  // ... tìm được [-1, 0, 1] lần nữa!
  result.push([...]);
  // → Duplicate results!
}

// ✅ Đúng: Skip duplicates
if (i > 0 && nums[i] === nums[i - 1]) continue;
while (left < right && nums[left] === nums[left + 1]) left++;
while (left < right && nums[right] === nums[right - 1]) right--;

// ❌ Pitfall 3: Không dừng khi nums[i] > 0
// 3 số dương không thể có tổng = 0
if (nums[i] > 0) break;

// ❌ Pitfall 4: Nhầm left/right boundaries
// left phải bắt đầu từ i + 1, không phải 0
// right phải bắt đầu từ n - 1
let left = i + 1;
let right = nums.length - 1;
```

### 🔄 5. Các hướng tiếp cận (So sánh)

#### 🔹 Cách 1: Brute Force (O(n³))

```javascript
function threeSum(nums) {
  const result = [];
  for (let i = 0; i < nums.length; i++) {
    for (let j = i + 1; j < nums.length; j++) {
      for (let k = j + 1; k < nums.length; k++) {
        if (nums[i] + nums[j] + nums[k] === 0) {
          result.push([nums[i], nums[j], nums[k]]);
        }
      }
    }
  }
  return result;
}
```

**📊 Phân tích:**
```
Time:  O(n³) — 3 nested loops
Space: O(1)
```

**⚠️ Nhược điểm:**
- Quá chậm cho n = 3000

---

#### 🔹 Cách 2: Hash Set (O(n²))

```javascript
function threeSum(nums) {
  const result = [];
  nums.sort((a, b) => a - b);
  
  for (let i = 0; i < nums.length; i++) {
    if (i > 0 && nums[i] === nums[i - 1]) continue;
    if (nums[i] > 0) break;
    
    const seen = new Set();
    for (let j = i + 1; j < nums.length; j++) {
      const complement = -(nums[i] + nums[j]);
      if (seen.has(complement)) {
        result.push([nums[i], complement, nums[j]]);
        while (j + 1 < nums.length && nums[j] === nums[j + 1]) j++;
      }
      seen.add(nums[j]);
    }
  }
  return result;
}
```

**📊 Phân tích:**
```
Time:  O(n²)
Space: O(n)
```

#### 🔹 Cách 3: Sort + Two Pointers (O(n²)) ⭐ **TỐI ƯU**

```javascript
function threeSum(nums) {
  const result = [];
  nums.sort((a, b) => a - b);
  
  for (let i = 0; i < nums.length; i++) {
    if (i > 0 && nums[i] === nums[i - 1]) continue;
    if (nums[i] > 0) break;
    
    let left = i + 1;
    let right = nums.length - 1;
    
    while (left < right) {
      const sum = nums[i] + nums[left] + nums[right];
      
      if (sum === 0) {
        result.push([nums[i], nums[left], nums[right]]);
        
        while (left < right && nums[left] === nums[left + 1]) left++;
        while (left < right && nums[right] === nums[right - 1]) right--;
        
        left++;
        right--;
      } else if (sum < 0) {
        left++;
      } else {
        right--;
      }
    }
  }
  
  return result;
}
```

**📊 Phân tích:**
```
Time:  O(n²) — sort O(n log n) + 2Sum O(n)
Space: O(1)
```

**✅ Ưu điểm:**
- Không dùng extra memory
- Skip duplicates dễ dàng
- Fast

// Variation 1: 3Sum Closest (LeetCode 16)
// Tìm bộ 3 có tổng GẦN NHẤT với target
function threeSumClosest(nums, target) {
  nums.sort((a, b) => a - b);
  let closest = Infinity;
  
  for (let i = 0; i < nums.length; i++) {
    let left = i + 1, right = nums.length - 1;
    while (left < right) {
      const sum = nums[i] + nums[left] + nums[right];
      if (Math.abs(sum - target) < Math.abs(closest - target)) {
        closest = sum;
      }
      if (sum < target) left++;
      else right--;
    }
  }
  return closest;
}

## ✅ Đáp án cuối cùng

```javascript
/**
 * 3Sum - Sort + Two Pointers
 * 
 * Ý tưởng:
 * 1. Sort mảng
 * 2. Với mỗi nums[i], tìm 2 số có tổng = -nums[i]
 * 3. Dùng Two Pointers cho Two Sum
 * 4. Skip duplicates
 * 
 * Time: O(n²) | Space: O(1)
 * 
 * @param {number[]} nums
 * @return {number[][]}
 */
var threeSum = function(nums) {
  const result = [];
  nums.sort((a, b) => a - b);
  
  for (let i = 0; i < nums.length; i++) {
    // Skip duplicates cho i
    if (i > 0 && nums[i] === nums[i - 1]) continue;
    
    // Dừng nếu nums[i] > 0
    if (nums[i] > 0) break;
    
    let left = i + 1;
    let right = nums.length - 1;
    
    while (left < right) {
      const sum = nums[i] + nums[left] + nums[right];
      
      if (sum === 0) {
        result.push([nums[i], nums[left], nums[right]]);
        
        // Skip duplicates cho left và right
        while (left < right && nums[left] === nums[left + 1]) left++;
        while (left < right && nums[right] === nums[right - 1]) right--;
        
        left++;
        right--;
      } else if (sum < 0) {
        left++;
      } else {
        right--;
      }
    }
  }
  
  return result;
};
```
