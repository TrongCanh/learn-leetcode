Tree (Cây Phân Cấp)

Tuyệt kỹ quản lý dữ liệu phân cấp. Sân chơi của Đệ quy (Recursion) và Divide & Conquer.
🔥 Insight: Nghĩ về Tree là nghĩ về [Root] + [Subtree Trái] + [Subtree Phải]

Lý thuyết Trọng tâm

Bản chất: Recursion (Đệ quy)

Tree không phải cấu trúc tuyến tính. Mỗi Node trong Tree bản thân nó lại là Gốc (Root) của một Tree nhỏ hơn bên dưới. Giải bài toán Tree là giải bài toán nhỏ ở nhánh Trái/Phải rồi tổng hợp lại.

Pre-order Root ➔ Left ➔ Right
In-order Left ➔ Root ➔ Right
Post-order Left ➔ Right ➔ Root

Các Pattern Cốt Lõi

  • 1 DFS Traversal: Duyệt sâu. Đi sâu hết cỡ xuống 1 nhánh lá rồi mới quay lại (Backtrack). Dùng Stack của hệ thống (Đệ quy).
  • 2 Divide & Conquer: Trả về kết quả từ dưới lên. Hàm ở Node Cha sẽ chờ kết quả từ dfs(left) và dfs(right) để tính toán tiếp (VD: Tính độ sâu Max Depth).
  • 3 Binary Search Tree (BST): Node Trái < Node Cha < Node Phải. Tra cứu/Chèn/Xóa siêu tốc độ O(log n). Việc duyệt In-order trên BST sẽ cho ra mảng ĐÃ SẮP XẾP.
